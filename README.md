# Estándares ABAP ICASA

Este repositorio contiene una versión del documento de estándares y nomenclaturas de desarrollo ABAP de ICASA, preparada para su uso en GitHub.

## ¿Qué incluye?

- Estándares de desarrollo ABAP
- Reglas de nomenclatura
- Lineamientos generales de codificación
- Buenas prácticas de rendimiento (performance)
- Criterios de robustez y mantenibilidad
- Instrucción general para asistencia con IA en desarrollo ABAP (alineada a Clean ABAP / SOLID)

## Archivo principal

- [Ver estándares ABAP](Estandares.md)

## Propósito

Centralizar en un formato versionable y fácil de consultar los lineamientos técnicos del equipo para desarrollos ABAP, refactor, revisiones de código y trabajo asistido por IA.

## Uso recomendado

- Consultar el archivo antes de iniciar desarrollos nuevos
- Usarlo como referencia en revisiones técnicas
- Mantenerlo actualizado cuando cambien estándares del equipo
